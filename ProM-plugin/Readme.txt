Please follow the following steps to use the plug-in "DSE.jar ":

1. Put the DSE.jar File into the "extraLib" file of the ProM project, and then add jar file to the build path (cf. g1.png).

2. Run the ProM project and you can search the "DSE Plug-in" (cf. g2.png).

3. Choose the correct file and start the plug-in, you will get the results (cf. g3.png and g4.png) .

Note: You can use the log file in the "demo" folder as the input to test the DSE plug-in.
